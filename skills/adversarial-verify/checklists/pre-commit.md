# Pre-commit checklist
Read this immediately before the user books / buys / signs. Do not skip on the grounds that you checked earlier — **you checked an earlier moment.**

- [ ] **Identity.** The final URL (post-redirect) resolves to the exact item. ID / title / coordinates match what we recommended.
- [ ] **Availability.** Confirmed live, in the channel the user will actually use, for the exact parameters.
- [ ] **Price.** All-in, recomputed today: base × quantity + every fee + tax. Under the ceiling. Basis stated.
- [ ] **Refundability.** Cancellation terms read and reported. If the quoted price requires a non-refundable rate, say so explicitly.
- [ ] **The headline claim.** Re-checked against an independent source, not the listing.
- [ ] **The missing column.** Still holds for this specific item.
- [ ] **Parameters unchanged** since the numbers were gathered. If they changed, every number is unverified.
- [ ] **Artifact matches reality.** No section still recommends a ruled-out option.
- [ ] **Tradeoffs named**, then hand off. The human clicks the button. Never complete a booking, payment, or form.
